Divide and Conquer


Q: So how do I know if I need to divide and conquer? This Question is easy!
A: If the problem is easily solved, then solve it, but if the problem is difficult, or seemingy impossible, then simply break it in to smaller parts.

Q: Now I need to know how to divide and conquer.
A: Break the huge problem up into several parts! Check out the example below!


You need to find a phone number in a book of 1000 numbers, the book is organized by name alphabetically.
  1. First you need to find out the name assoicated with the number
  2. Then flip the book to a estimated region
  3. Start narrowing down the name and number on a page.
  4. Select the number
